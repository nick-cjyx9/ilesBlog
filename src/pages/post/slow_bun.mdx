由于设置代理后 Bun 会报错，只能全局换成国内源。

<Excerpt />

```powershell
notepad $HOME\.bunfig.toml
```

新建文件 -> 粘贴以下内容

```toml
[install]
registry = "https://registry.npmmirror.com/"
```
