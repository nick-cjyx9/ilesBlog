# Recall·0x00:退出信竞

<span class="w-6 inline-block"></span>2023年对于我来说是一个意义非凡的年份，来到一线城市上学、参加信息竞赛、第一次被记“严重违纪”，过去的
半年改变了我特别多。进入这所高中之前，我在四五线城市的普通中学中度过了相对舒适和 ~~成功？~~ 的三年，在初中毕业之前，我似乎从来都是以一个 **“好学生”** 的
形象生活着，凭借成绩，可以很容易的维持这种形象，所以才会觉得自己无忧无虑吧。现在和她聊起初中时，我竞不知道初中三年每天是如何在睡眠不足的情况下**保持动力**的，
现在想起来，还蛮羡慕曾经对学校富有激情的自己的。

<span class="w-6 inline-block"></span>编程的爱好其实早在六年级就开始，只是初中三年没有丝毫进步就是了，<Heimu>突然很庆幸自己那个充满黑历
史的Github号被封了， ~~虽然现在的号提交记录也好不了哪里去~~ </Heimu> 稀里糊涂地到了高中，听说学校在找信息竞赛生（OIer），我也是自信于自己的一点点基础，去报
了名，记得最初的机房人几乎**坐满了**，我们被分成无基础和有初中基础两部分，而我在无基础组内的水平又算是比较出众，所以学起来干劲十足。过了大概两个月，经历了几次文化课
考试，机房走了很多人（基本都是在考完出分的节点退出），因为我的文化课也考的不是很理想，父母也数次表达出他们的不满，但我依然不想随便放弃。又这样继续了两个月，差不多开始学图论时，机房里从最初坐满到只剩
下两只“菜鸟”了。我也慢慢意识到：这个学校几乎**一点也不重视OI**，没有练题时间，没有资源，啥也没有，外加文化课施压，所以在这里**省1≈做梦**。<Heimu>这是一个才学到dijkstra的蒟蒻该说的吗(doge)</Heimu>文化课的不如意
，和有基础的大佬的差距，家里的、老师的压力，让我的状态变得不太好，在训练时频繁摸鱼，学啥都提不起劲，再加上受焦虑风气的影响。那段时间，我几乎快疯了，但还是撑着，
直到一次晚上打牌被抓后，班主任毫不客气地给我记了“严重违纪”，并要求见家长，一通电话，让我下了一半QuitOI的决心，几个不眠之夜后，我做出了决定，因为OI或许已经**不能为我带来什么了**，吧。

<span class="w-6 inline-block"></span>在这之后，文化课成绩算是有所长进，但还是会数次想起曾经欢笑的牌桌，还有一起摸鱼，看电影的集训时光。
事到如今，我仍旧会觉得：QuitOI是不是最好的选择呢？OI的确给了我许多欢乐，和一条出路。
迷茫时，我看了不少Blog（ 像是 https://ouuan.moe ,羡慕大佬精彩的大学生活，~~应该不会被本人看到~~ ），听了一些Podcast，开始漫长的思考，
慢慢地找回了自己的一点激情，但有时候也禁不住会想：当初再努力一点，我是不是也可以在OI干出成绩？我真不明白。
