# Recall·0x00:退出信竞

<span class="w-6 inline-block"></span>2023年对于我来说绝对是意义非凡的一年，来到一线城市上学、参加信息竞赛、人生中第一次被记“严重违纪”，过去的
半年改变了我特别多。进入这所高中之前，我在四五线城市的普通中学中度过了相对舒适和 ~~成功？~~ 的三年，在初中毕业之前，我似乎从来都是以一个 **“好学生”** 的
形象生活着，凭借成绩，我可以很容易的维持这种形象，所以才会觉得自己无忧无虑吧。现在和她聊起初中时，我竞不知道初中三年天天 **睡眠不足** 又是如何坚
持 **充满动力** 的，现在想起来，还蛮羡慕对学校富有激情的自己的。

<span class="w-6 inline-block"></span>编程的爱好其实早在六年级就开始，只是初中三年没有丝毫进步就是了，<Heimu>突然很庆幸自己那个充满黑历
史的Github号被封了， ~~虽然现在的号也好不了哪里去~~ </Heimu> 稀里糊涂地到了高中，听说学校在找信息竞赛（OI）生，我也是自信于自己的一点点基础，去报
了名，最初的机房里人**几乎坐满了**，分成无基础和有初中基础两部分，我在无基础组内的水平又算是比较出众，所以学起来干劲十足。可是逐渐地，经历了几次文化课
和算法考试，机房退出了很多人，我的父母也数次表达出OI对我文化课的影响，但我依然不想放弃。一边苦恼着自己，差不多开始学图论时，机房里从最初坐满到只剩
下两个“菜鸟”了。确实，我也慢慢意识到：这个学校**一点也不重视OI**，没有练题时间，没有资源，教练也似乎没有干劲（叠甲：我其实很感谢我的教练，她在我犯
错时、退出信竞时都给了我莫大的理解），所以在这里**省1≈做梦**。但如果要退出？我没脸退也不敢退，只因自己的偏执导致的当初和父母的争吵。文化课的不如意
，和有基础的dalao的差距，家里的老师的压力。让我寻求逃避，我开始在训练时摸鱼，学习时也提不起劲，再加上受互联网上焦虑文字的影响，我几乎快疯了，任由自
己逃避现实，在一次晚上打牌被发现后，班主任毫不客气地给我记了人生中第一个“严重违纪”，面对我这种状态，最先崩溃的**不是我**，是我的母亲，一通电话，让
我下了一半QuitOI的决心，几个不眠之夜后，我做出了决定，因为OI已经**不能为我带来什么了**，或许吧。

<span class="w-6 inline-block"></span>在这之后，文化课成绩算是有所长进，但还是会数次想起机房的“幼儿园”牌桌，还有一起摸鱼，看电影的集训时光。
但是事到如今，我仍旧会觉得：QuitOI是不是最好的选择呢？迷茫时，我看了不少Blog（ 像是 https://ouuan.moe ,每天都有事可做，有事可记的大学生活给了我
很多motivation，一直不敢去博客骚扰，但是还是十分感谢您，~~应该不会被本人看到~~ ），听了一些Podcast，也逐渐找回了自己对于编程的激情，这也引发了
我的反思：再努力一点，我是不是也可以在OI干出成绩？我很矛盾。<Heimu>这是一个只学到dijkstra的蒟蒻该说的吗(doge)</Heimu>

# Recall·0x01:住校时光

<span class="w-6 inline-block"></span>曾经因为集训九十多天没有回过老家一趟，最终阳了~~被迫~~以连续住校97天成绩遗憾离场，现在想来，
