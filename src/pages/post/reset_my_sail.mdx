# Recall·0x00:退出信竞

<span class="w-6 inline-block"></span>2023年对于我来说是一个意义非凡的年份，来到一线城市上学、参加信息竞赛、第一次被记“严重违纪”，过去的
半年改变了我特别多。进入这所高中之前，我在四五线城市的普通中学中度过了相对舒适和 ~~成功？~~ 的三年，在初中毕业之前，我似乎从来都是以一个 **“好学生”** 的
形象生活着，凭借成绩，可以很容易的维持这种形象，所以才会觉得自己无忧无虑吧。现在和她聊起初中时，我竞不知道初中三年每天是如何在睡眠不足的情况下**保持动力**的，
现在想起来，还蛮羡慕曾经对学校富有激情的自己的。

<span class="w-6 inline-block"></span>编程的爱好其实早在六年级就开始，只是初中三年没有丝毫进步就是了，<Heimu>突然很庆幸自己那个充满黑历
史的Github号被封了， ~~虽然现在的号提交记录也好不了哪里去~~ </Heimu> 稀里糊涂地到了高中，听说学校在找信息竞赛生（OIer），我也是自信于自己的一点点基础，去报
了名，记得最初的机房人几乎**坐满了**，我们被分成无基础和有初中基础两部分，而我在无基础组内的水平又算是比较出众，所以学起来干劲十足。过了大概两个月，经历了几次文化课
考试，机房走了很多人（基本都是在考完出分的节点退出），因为我的文化课也考的不是很理想，父母也数次表达出他们的不满，但我依然不想随便放弃。又这样继续了两个月，差不多开始学图论时，机房里从最初坐满到只剩
下两只“菜鸟”了。我也慢慢意识到：这个学校几乎**一点也不重视OI**，没有练题时间，没有资源，啥也没有，外加文化课施压，所以在这里**省1≈做梦**。<Heimu>这是一个才学到dijkstra的蒟蒻该说的吗(doge)</Heimu>文化课的不如意
，和有基础的大佬的差距，家里的、老师的压力，让我的状态变得不太好，在训练时频繁摸鱼，学啥都提不起劲，再加上受焦虑风气的影响。那段时间，我几乎快疯了，但还是撑着，
直到一次晚上打牌被抓后，班主任毫不客气地给我记了“严重违纪”，并要求见家长，一通电话，让我下了一半QuitOI的决心，几个不眠之夜后，我做出了决定，因为OI或许已经**不能为我带来什么了**，吧。

<span class="w-6 inline-block"></span>在这之后，文化课成绩算是有所长进，但还是会数次想起曾经欢笑的牌桌，还有一起摸鱼，看电影的集训时光。
事到如今，我仍旧会觉得：QuitOI是不是最好的选择呢？OI的确给了我许多欢乐，和一条出路。
迷茫时，我看了不少Blog（ 像是 https://ouuan.moe ,羡慕大佬精彩的大学生活 ），听了一些Podcast，开始漫长的思考，
慢慢地找回了自己的一点激情，但有时候也禁不住会想：当初再努力一点，我是不是也可以在OI干出成绩？我真不明白。

## 县城之人
<span class="w-6 inline-block"></span>未来，我可能再也没机会回到这里了吧。人总怕忘事，我和ta们的一些细节早已记不清楚，故在此记录一下还能记起的片段。

### 同学们
<span class="w-6 inline-block"></span>A是一个开朗的女生，和我从幼儿园开始小学、初中十多年都和我同校同班，<Heimu>从小学起，我就产生了对她的爱慕之情，</Heimu>，但是从6年级班上有
人在传“我喜欢A”的话时，我开始故意地远离她，我们之间的交流越来越少，6年级时还会在考试后讨论答案，上了市里的初中后，我们意外地被分到了一个班。
本以为能继续追求她，但似乎A对我没有表现出一点好感，没过多久，就有人传起小学时一样的话，于是我和A似乎越来越远。初中的前半部分时间，我很贪玩，
成绩却不是很差，就固步自封起来，没有了进取之心。那时，A的成绩还不如我。从初中第二年开始，A的成绩突飞猛进，她进入年级前十，我居然会感到羞愧和不甘。为此，我开始默默地以他为目标奋斗起来。
长期的隔绝让我以为我们两个不可能会有任何瓜葛，但是不知为何，同样地从县城来到城市，我不自觉会和A处处比较，会偷偷看她，会躲避她的眼神，但就是不敢
和他交流，可能我对A的认识还停留在小学，导致我不敢于也不知道以何种关系和她交流。3年很快就过去了，毕业的时候，A来和我合照，我的心脏那是砰砰直跳，
但在这之后，A和我又如同形同陌路。现在我们到了更大的城市，只是这次就没能这么幸运了。。。

<span class="w-6 inline-block"></span>原以为来到这里会慢慢地忘掉A，但每个孤单的晚上，我还是会想，还是会想起小学时一起来到成都的经历，为
数不多的和她的美好记忆，明明已经不可能了，为什么会一直想呢。听着她的偶像：周杰伦的歌，我越来越搞不懂一切。现在想起A，和她发生在县城的事还历历在目，
尽管许多遗憾，但至少我们如今各赴前程，已经成功地 **“从大山走了出来”**。

### 长辈们
<span class="w-6 inline-block"></span>在我一路更换城市读书时，我的家人们给了我莫大的帮助，往常我的姥爷还在时，一大家人总会在节日聚在一起。
直到他去世之后，这种场景就**再也没有过**。我的姥爷生前十分宠爱我，他的离去也是我第一次有意识地经历至亲之人死亡。现在想起来，我还是会觉得悲痛、恐惧。
现在，他已经离去了好几年了，我也慢慢开始忘记和他的事情。

- 我走上编程这个道路完全是因为我的姥爷，我的父母对于我上网管的十分严格，但我一有空就溜去姥爷的屋子偷偷上网，podcast《代码之外》第二期中，Evan说过
越早找到一个孩子的爱好就越幸运，因为姥爷对我的包容，我开始接触、学习一些代码，并爱上了编程。
- 姥爷毫不吝惜地让我花钱买零食，这一点至今仍心存愧疚
- 姥爷曾经是个军人，他常常给我讲他之前在炊事营的见闻和曾经时如何作为代表来到人民大会堂见毛主席的事情，这些事熏陶着我，但我当初并没有好好听，细节
已经不记得一点了，深感愧疚。
- 我上初中后，就很少去见他了，那时候正赶上疫情，或许是老人最需要人陪伴的孤独时期吧，我奶奶走的早，一直都是姥爷**一个人生活**。但是我当时竟然没有好好陪陪
他。那时，姥爷的身体状况一天不如一天，甚至会开着电视睡着，烟头落下把自己烫伤。一天，大概是疫情最严重的21年初，姥爷提着一堆生活用品和食物来看我，只是短短
地询问了我在城市的学习生活，教导了几句就离开了，我仍记得他说了一句：“我有一天也要去**见马克思**了”，我只是强忍着没有落泪，无言可答。

### 可悲之人

<span class="w-6 inline-block"></span>曾经的县城，虽然没有脱贫，但仍有很多有梦想的人在。现在的县城，虽然有了更大的学校，可我看到的是学生
无心学习，抽烟酗酒，我知道这一切不怪他们，但一想到这些人以后的命运可能已经被定了十有八九，就感到十分可悲。
