### 关于这个博客

- 某个暑假闲来无事，照着大佬的博客 [ouuan's blog](https://ouuan.moe/)，用 iles 半抄半 Google 地做了这个博客，并且注册了`nickchen.top`的域名。

- 设计理念：**不原生地使用任何圆角，少用阴影**

- 博客框架使用 [iles](https://iles.pages.dev)（不建议使用[^1]，正在考虑迁移到 astro ），UI 使用 tailwind css ，图标字体用 [fontello](https://fontello.com/) 打包，博客主要使用霞鹜文楷和 Noto Serif 字体，统计方面使用跑在 vercel + supabase 上的 umami 实例。

[^1]: iles 目前开发非常不活跃，不建议开发新项目时选用 iles。

- 博客部署在 Cloudflare Pages 上。

- 最初的代码是 Nick 1边从0开始学 Vue 1边写的，可读性和逻辑各种方面都极差，虽然代码已经开源在[Github·nick-cjyx9/ilesBlog](https://github.com/nick-cjyx9/ilesBlog)，但是尽量**不要**大量参考。
