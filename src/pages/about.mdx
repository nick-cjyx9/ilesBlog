### 关于这个博客

- 某个暑假闲来无事，照着大佬的博客 [ouuan's blog](https://ouuan.moe/)，用 iles 半抄半 Google 地做了这个博客，并且注册了 `nickchen.top` 的域名。

- 博客框架使用 [iles](https://iles.pages.dev)（不建议使用[^1]，最近喜欢上  [Astro](https://astro.build)，并准备把博客用 Astro 重写），UI 使用 tailwind css ，图标字体用 [fontello](https://fontello.com/) 打包，博客主要使用霞鹜文楷和 Noto Serif 字体，统计方面使用跑在 vercel + supabase 上的 umami 实例。

[^1]: iles 目前开发非常不活跃，不建议开发新项目时选用 iles。

- 博客由 Github Actions 编译并部署在 Cloudflare Pages 上。

- 最初的代码是 Nick 1 边从 0 开始学 Vue 1边写的，可读性和逻辑各种方面都极差，虽然代码已经开源在 [Github·nick-cjyx9/ilesBlog](https://github.com/nick-cjyx9/ilesBlog)，但是尽量**不要**大量参考。
